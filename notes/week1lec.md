## Course Description / 课程描述  
This course provides a theoretical account of consumption as a **cultural activity**. Consumers interact with each other, the market, and socio-cultural institutions through consumption. Consumption is both an individual activity for satisfying needs and an aggregate cultural phenomenon. This course reviews consumer culture and consumer behaviour theories to understand their implications for marketers.  
本课程从理论角度探讨消费作为一种**文化活动**，消费者通过消费行为与他人、市场和社会文化机构互动。消费既是一种满足个人需求的活动，也是一种集体文化现象。本课程将回顾消费者文化和消费者行为理论，以理解其对市场营销的影响。  

### Course Learning Outcomes / 课程学习目标  
1. Identify relevant consumer culture and behaviour theories / 识别相关的消费者文化和行为理论  
2. Examine theories in individual, marketplace, and cultural contexts / 在个人、市场和文化背景下分析相关理论  
3. Relate consumption experiences to consumer culture and behaviour theories / 将消费体验与消费者文化和行为理论联系起来  
4. Critique theories in different contexts / 评估不同背景下的理论  
5. Demonstrate the use of evidence and theory in marketing strategies / 运用证据和理论制定市场营销策略  
6. Reflect on personal consumption experiences / 反思个人消费体验

## Theoretical Perspectives on Consumption / 消费的理论视角  
### Why Do We Consume? **Value!** / 为什么我们消费？**价值！**  
- **Exchange value / 交换价值**: Market transactions / 市场交易  
- **Value-in-use / 使用价值**: Consumption experience / 消费体验  
- **Symbolic value / 符号价值**: Identity expression / 通过消费表达身份  
- **Value co-creation / 价值共创**: Collaboration between producers and consumers / 生产者和消费者之间的协作  

### **Theoretical Perspectives / 理论视角**  
- **Economic perspective / 经济学视角**: Value destruction by consumers / 认为消费者会消耗价值  
- **Symbolic perspective / 符号学视角**: Product meanings transfer value / 产品象征意义带来价值  
- **Consumer culture perspective / 消费文化视角**: Consumption is culturally embedded / 认为消费嵌入在文化体系中

## Consumer Research Perspectives / 消费者研究视角  
### Philosophical Perspectives / 哲学视角  
- **Positivism (Modernism) / 实证主义（现代主义）**  
- **Interpretivism (Postmodernism) / 解释主义（后现代主义）**  

### Consumer Research: Positivism vs. Interpretivism / 消费者研究：实证主义 vs 解释主义  
- **Positivism / 实证主义**: Structured, objective study of consumer behavior / 结构化、客观地研究消费者行为  
- **Interpretivism / 解释主义**: Subjective, meaning-driven consumer insights / 主观的、意义驱动的消费者见解

## Consumption Types / 消费类型  
### How We Consume / 我们如何消费  
- **Consuming as an experience / 体验式消费**: Emotional or aesthetic reactions / 情感或美学反应  
- **Consuming as integration / 整合式消费**: Expressing identity through consumption / 通过消费表达身份  
- **Consuming as classification / 分类式消费**: Associating with objects socially / 通过物品进行社交归属  
- **Consuming as play / 娱乐式消费**: Participating in shared consumer experiences / 参与共享的消费体验  

**Example / 例子:** Gaming Consumption / 游戏消费  
- **Experience / 体验**: Playing the game and enjoying it / 玩游戏并享受过程  
- **Integration / 整合**: Customizing an avatar, choosing storylines / 自定义角色、选择剧情  
- **Classification / 分类**: Using game-related images on social media / 在社交媒体使用游戏相关图片  
- **Play / 娱乐**: Interacting in online communities / 在网络社区交流互动

## Marketing Ethics / 市场营销伦理  
### Influence vs. Manipulation / 影响 vs 操纵  
- **Do marketers create artificial needs? / 市场营销是否制造了人为需求？**  
  - *Need / 需求*: A biological necessity / 生理必需  
  - *Want / 欲望*: A learned way to satisfy that need / 满足需求的学习方式  
- **Are advertising and marketing necessary? / 广告和营销是否必要？**  
  - Marketing provides consumer information / 营销提供消费者信息  
- **Do marketers promise miracles? / 营销是否承诺奇迹？**  
  - Advertisers lack total control over consumer behavior / 广告商无法完全控制消费者行为  

### Consumer Responsibility / 消费者责任  
- **Buyer Beware vs. Seller Regulation / 买方自负责任 vs 卖方监管**  
- **Consumer Responsibilisation / 消费者责任化**: A neoliberal governance approach that shifts responsibility for social problems from institutions to individuals. / 一种新自由主义治理方式，将社会问题的责任从机构转移到个人。  
  - *Example / 例子:* Carbon footprint calculators place responsibility on consumers rather than corporations. / 碳足迹计算器将责任推给消费者，而非公司。
